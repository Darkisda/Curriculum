Gosto bastante de fazer meus códigos em JavaScript e em Java. Porque essas duas?

JavaScript é uma das linguagens mais utilizadas no mercado, sendo utilizada na maioria dos apps mobile. Uma das principais caracteristicas do JavaScript é o fato dela ser fracamente tipada, ou seja, você nao precisa explicar o tipo daquela variável que o prórpio interpretador do JavaScript atribui um tipo a ela.

Java, como muitos pensam, não é parecido com JavaScript. Java é uma linguaguem de programação fortemente tipada, ou seja, você tem que descrever bem os tipos e métodos que você deseja usar, deixando assim o seu código bem robusto, e dificil de quebrar.
